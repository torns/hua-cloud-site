# 介绍

[vue-hua-cloud-template](https://github.com/hua-cloud/cloud-template) 是基于 Vue2.6+, Vue-cli3.x, Element UI的一个前端管理后台集成解决方案。它使用了最新的前端技术栈, 强化统一的编码规范，提炼了典型的业务模型，它可以作为公司统一的中后台项目模板。

## 功能

- 代码校验
- 样式校验
- commit校验
- 编辑器配置
- 代码生成
- 注释生成文档
- 自动加载
